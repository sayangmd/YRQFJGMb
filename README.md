# 项目简介

## 核心功能
- 基于YOLOv8实现老师课堂行为检测，支持6类行为识别：Crossing legs、Using a phone、Teaching or asking、Looking at the screen、Writing、Guiding students
- 提供多种输入方式支持：单张图片、图片文件夹、视频文件、本地摄像头
- 实时显示检测结果：目标位置、目标总数、置信度等信息
- 支持参数调整：自定义置信度、IOU阈值
- 提供显示选项：选择是否显示标签和原图
- 支持模型切换功能
- 批量检测并查看所有结果
- 支持检测结果保存

## 适用场景
- 教育机构对教师课堂行为的分析研究
- 教学质量评估与改进
- 课堂行为数据收集与分析
- 教育研究中的行为模式识别

## 技术特点
- 技术栈：Python 3.9、OpenCV、PyQt5、Ultralytics、PyTorch 1.9
- 使用YOLOv8模型进行目标检测
- 包含3400张训练数据集
- 提供完整UI界面源文件及图标
- 配置好环境后可直接使用
